## External Dependencies

While the project is not based on a previous engine, it does have third party dependencies:

 - `OpenGL 4.5`
 - `GLFW 3.3.2`
 - `GLM 9.9.7`
 - `ImGui 1.77`
 - `stb_image v2.26`
 - `glad 0.1.33`
